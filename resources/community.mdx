## Report Issues

Found a bug or have a feature request?

<Steps>
  <Step title="Check existing issues">
    Search the [GitHub issues](https://github.com/buildermethods/design-os/issues) to see if someone else has reported the same problem or requested the same feature.
  </Step>
  
  <Step title="Open a new issue">
    If you don't find an existing issue, [create a new one](https://github.com/buildermethods/design-os/issues/new) with:
    - Clear description of the problem or feature request
    - Steps to reproduce (for bugs)
    - Expected vs. actual behavior
    - Your Design OS version number
  </Step>
  
  <Step title="Provide context">
    The more detail you provide, the faster the issue can be resolved. Include relevant file contents, error messages, or screenshots if helpful.
  </Step>
</Steps>
